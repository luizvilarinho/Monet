# Task: Agenda — Related Content, Embed de Notas e RAG

## Objective
Realizar três melhorias na funcionalidade de agenda do Monet: (1) remover o related content das notas de agenda, pois o título é uma data e a indexação retorna outras notas de agenda de forma inútil; (2) permitir que qualquer nota do app seja incorporada/visualizada dentro de uma data da agenda; (3) habilitar acesso ao RAG (busca na knowledge base) nas notas de agenda.

## Context
- Módulo de agenda (Calendar notebook) — notas com campo `date`, ID persistido em localStorage
- Sistema de Related Content — indexação de notas por similaridade semântica/título
- Sistema RAG — busca vetorial na knowledge base (`monet-vec.db`)
- Chat de notas — interface onde o RAG é consultado

## Requirements
- **[1] Remover Related Content da agenda:** Notas do tipo agenda (calendar notebook) não devem exibir o painel de related content. A indexação pode continuar existindo para outros notebooks, mas não deve aparecer na UI da agenda.
- **[2] Embed de notas na agenda:** Dentro de uma data da agenda, o usuário deve poder buscar e incorporar qualquer nota do app (de qualquer notebook). A nota incorporada deve ser visualizável inline na data.
- **[3] RAG na agenda:** As notas de agenda devem ter acesso ao mesmo mecanismo de busca RAG disponível para outras notas (consulta à knowledge base ao usar o chat).

## Constraints
- Não quebrar o related content dos demais notebooks
- O embed de notas deve ser uma referência/link para a nota original, não uma cópia
- O RAG da agenda deve seguir o mesmo padrão já implementado nas outras notas
- Não alterar o schema do banco de dados se possível; preferir soluções via UI/lógica

## Edge Cases
- Nota de agenda incorporando outra nota de agenda (loop visual deve ser evitado ou tratado)
- Nota incorporada que foi deletada — exibir estado de "nota não encontrada"
- RAG sem documentos na KB — comportamento deve ser o mesmo das outras notas (mensagem apropriada)
- Related content de uma nota normal que referencia uma nota de agenda — não deve ser afetado pela mudança
