## src/components/RelatedContent/scoreNotes.ts

**Correção:** Correto para o caso direto. A bidirecionalidade está implementada: se A embute B (`activeLinkedIds.has(note.id)`) ou B embute A (`noteLinkedIds.has(active.id)`), ambos aparecem no related content um do outro.

**Problema de performance — `[importante]`:** Para cada nota candidata, `extractLinkedNoteIds(note.content)` é chamado dentro do loop. Se o usuário tiver centenas de notas com conteúdo HTML longo, isso executa uma regex sobre todo o conteúdo de cada nota a cada re-render do RelatedContent. O custo é O(n × tam_conteúdo). O padrão atual de `scoreRelatedNotes` já faz isso para `tokenize`/`cleanContent`, mas a regex de embed é adicional. Dado que o projeto ainda está em fase de testes sem usuários ativos, o impacto prático é baixo por ora, mas vale registrar.

**Problema de topK com linked notes — `[nit]`:** Notas linked recebem `score: 1.0` e entram no array como as primeiras posições após o `.sort()`. Se houver mais de `topK` notas linked, as notas de alta relevância Jaccard serão descartadas pelo `.slice(0, topK)`. O comportamento resultante é que notas linked "empurram" notas relevantes para fora do painel. Pode ou não ser a intenção — não há instrução explícita no task sobre isso. Se a intenção é que todas as notas linked apareçam sempre, o `topK` deveria ser aplicado somente às não-linked.

**Efeitos colaterais:** O early-return de empty-check agora inclui `activeLinkedIds.size === 0`. Isso é correto: uma nota sem tags, título ou conteúdo mas com embeds continuará gerando related content — comportamento desejado.

**Padrões:** `extractLinkedNoteIds` foi exportada, seguindo o padrão de `cleanContent` (também exportada). Consistente.

**Segurança:** A regex `data-note-id="([^"]+)"` não usa `ReDoS`-suscetível — o quantificador é sobre `[^"]+` (negated char class), que não causa backtracking catastrófico.

**Organização:** A constante `EMBED_SCORE = 1.0` como variável de módulo é desnecessária — valor usado uma única vez, sem plano de variação. `[nit]`
