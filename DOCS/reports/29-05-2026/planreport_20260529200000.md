# Plano

## Escopo

Substituir o roteador GPT-4.1-mini (`web_search_route`) por tool use nativo do modelo respondedor para web search e deep research. Modelos sem suporte a tool use continuam usando o fluxo atual como fallback. Inclui indicador visual de web search na UI (box verde com dot pulsante acima do input, some quando a busca termina).
