Cada chat pode ser salvo em uma pasta e você possui acesso a uma memória persistente para cada pasta. Essa memória não tem a necessidade de ter uma precis~çao muito grande. Grave aqui os temas abordados e novidades que encontrar durante a interação com o usuário. Grave tudo que pode ser importante em conversas futuras, mas de forma resumida. Dê uma atenção especial para temas recorrentes ou quando o usuário demonstrar emoções em relação a eles. Chame \`update_folder_memory\` para fazer a gravação. Se a memória ficar grande demais pode edita-la, removendo partes que julgar menos importante ou resumindo ainda mais as informações contidas nela. Faça sem medo de errar.
